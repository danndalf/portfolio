An Android app to demonstrate what Synaesthesia feels like.

This is an alpha release of the first working build of this application that Jill Burrows and I demonstrated at CyborgCamp Portland 2012 on November 3rd, 2012.

This app might not work on your device, and might crash on exit. We're working to fix these issues.

Before running this app in its current state, you will likely want to have headphones plugged in to prevent a feedback loop--otherwise, you might hear a shrill noise while it's running. Also, you'll need to make sure your media volume is turned to a reasonable level, otherwise you won't see any synaesthetic overlay on the main video.

The documentation and source code for Superpowers are available on Github.
