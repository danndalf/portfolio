Host static web content anonymously and securely.

When you need to publish data anonymously, such as to circumvent censorship by governments or corporations, The Tor Project provides tools that can help maintain your anonymity and safety. This software bundle sets up tor and a web server on your computer (Linux or OS X right now, Windows to be tested soon) to allow you to publish information without fear of censorship.

Once you have shell access to a server with some basic tools installed (such as bash and gcc), this will build your hidden service in-place.

Open Source Anonymity

  • No root access needed
  • Can run within a truecrypt volume
  • Entirely Open Source Software
