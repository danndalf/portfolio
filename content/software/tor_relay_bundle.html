Setup a new Tor relay in minutes

Relays are the heart of the Tor network, and every bit of bandwidth counts. If you want to run a non-exit Tor relay, this code can set that up for you in minutes.

  • No root required
  • Perl, curl, OpenSSL, and GCC
  • Running in around ten minutes

Setup is simple:

1. Create a directory for the relay to run in:

mkdir ~/tor_relay
cd ~/tor_relay

2. Download this installer:

curl -O https://raw.github.com/danndalf/tor_relay_bundle/master/setup.pl

3. Run this installer:

perl setup.pl
